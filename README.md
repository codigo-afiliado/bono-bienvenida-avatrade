## ❓ Preguntas Frecuentes (FAQ)

### 1. ¿Cuánto dinero necesito depositar para recibir el bono?
El depósito mínimo es **$100 USD**. Con $100 recibes $20 de bono (20%).

### 2. ¿El bono es dinero real o solo para operar?
El bono se acredita como **crédito de trading**. Puedes usarlo para operar y retirar las ganancias generadas, pero el bono en sí tiene requisitos de volumen para convertirlo en dinero retirable.

### 3. ¿Puedo retirar el bono directamente?
No. El bono se usa como margen adicional para operar. Las ganancias que generes con el bono son retirables. Consulta los términos de rollover de AvaTrade.

### 4. ¿Funciona en todos los países?
Funciona en la mayoría de países donde opera AvaTrade. Verifica la disponibilidad en tu jurisdicción durante el registro. Algunos países tienen restricciones regulatorias.

### 5. ¿Puedo combinar este bono con otras promociones?
Generalmente no. Los bonos de bienvenida son excluyentes con otras promociones. Revisa los términos al registrarte.

### 6. ¿Qué pasa si olvido introducir el código 128979 al registrarme?
No hay forma de añadirlo después del registro. Tendrías que crear una cuenta nueva. Por eso es importante **expandir el campo oculto** antes de enviar el formulario.

### 7. ¿Este código funciona en AvaTradeGO (app móvil)?
Sí, el código **128979** funciona tanto en la versión web como en AvaTradeGO. El campo oculto también está disponible en la app.

### 8. ¿Cada cuánto verifican que el código sigue activo?
Actualizamos este repositorio **mensualmente**. Si el código deja de funcionar, se notificará aquí inmediatamente con una advertencia visible.

### 9. ¿Puedo depositar varias veces para acumular más bono?
Sí. El bono máximo acumulable es **$14,000 USD**. Puedes hacer depósitos sucesivos y cada uno generará su bono correspondiente hasta alcanzar el tope.

### 10. ¿Hay límite de tiempo para usar el código después del registro?
AvaTrade no suele imponer un plazo estricto, pero recomendamos depositar dentro de los **primeros 30 días** tras la creación de la cuenta para asegurar el bono en las condiciones actuales.
